# Screenshots

Place product screenshots in this folder and keep filenames stable so the root README renders correctly.

Recommended exports:

- PNG
- width: 1170px (iPhone 15 Pro portrait) or consistent width across images
- include both Light and Dark mode when relevant

Suggested filenames (used by the README):

- `login.png`
- `home.png`
- `dashboard.png`
- `profile.png`
- `chat.png`
- `quiz.png`
